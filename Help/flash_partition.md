## FLASH 分区结构

| F000h - F3FFH   | Data Flash（EEPROM） | 1K   |
| :-------------- | :------------------- | :--- |
| 0xf000 - 0xf040 | 功能 模式 设置       | 64B  |
| 0xf040 - 0xf080 | LED灯光组            | 64B  |
| 0xf080 - 0xf0c0 | 按键 1 功能          | 64B  |
| 0xf0c0 - 0xf100 | 按键 2 功能          | 64B  |
| 0xf100 - 0xf140 | 按键 2 功能          | 64B  |
